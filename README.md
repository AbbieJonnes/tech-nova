# TECH-NOVA ELECTRONICS STORE

## Project Overview
Tech-Nova is a simple multi-page e-commerce website built using HTML, CSS, and Font Awesome icons.  
It showcases electronic products such as laptops, smartphones, and headphones with a clean and user-friendly layout.

## Project Structure

index.html        # Home page  
product.html      # Products page  
about.html        # About page  
contact.html      # Contact page  

index.css         # Home page styles  
product.css       # Products page styles  
about.css         # About page styles  
contact.css       # Contact page styles  

script.js         # JavaScript for interactions (e.g. dark mode toggle)  

assets/images/    # All project images  

## Features
- Responsive navigation bar  
- Product listing with categories  
- Filter section (Sort & Category)  
- About Us page with company description  
- Contact form page  
- Social media links  
- Dark mode toggle (optional enhancement)  
- Clean and simple UI design  

## Technologies Used
- HTML  
- CSS  
- JavaScript (for dark mode toggle)  
- Font Awesome (icons)
